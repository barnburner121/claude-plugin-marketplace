# log-aggregator

Generate log aggregation configs (ELK, Loki).

## Installation

```
/plugin install log-aggregator@barnburner121
```

## Usage

Once installed, the `log-agg` skill will be available in Claude Code. It provides tooling to Generate log aggregation configs (ELK, Loki).
