# insomnia-gen

Generate Insomnia workspace configs.

## Installation

```
/plugin install insomnia-gen@barnburner121
```

## Usage

Once installed, the `insomnia` skill will be available in Claude Code. It provides tooling to Generate Insomnia workspace configs.
