# i18n-extractor

Extract strings and set up internationalization.

## Tools Available

- **Read** — Read files from the filesystem
- **Write** — Write files to the filesystem
- **Edit** — Make targeted edits to existing files
- **Bash** — Execute shell commands
- **Grep** — Search file contents with regex
- **Glob** — Find files by pattern matching

## Usage

Invoke the `i18n` skill to Extract strings and set up internationalization. The skill will analyze the relevant codebase context and generate appropriate output.
