# slack-bot

Generate Slack bot with slash commands.

## Installation

```
/plugin install slack-bot@barnburner121
```

## Usage

Once installed, the `slack-bot` skill will be available in Claude Code. It provides tooling to Generate Slack bot with slash commands.
