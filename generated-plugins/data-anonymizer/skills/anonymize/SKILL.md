# data-anonymizer

Generate data anonymization scripts.

## Tools Available

- **Read** — Read files from the project
- **Write** — Write new files to disk
- **Edit** — Edit existing files in place
- **Bash** — Run shell commands
- **Grep** — Search file contents with regex
- **Glob** — Find files by pattern

## Usage

Invoke the `anonymize` skill to Generate data anonymization scripts. The skill analyzes your project structure and generates the appropriate code and configuration files.
