# stripe-setup

Generate Stripe integration with checkout and subscriptions.

## Installation

```
/plugin install stripe-setup@barnburner121
```

## Usage

Once installed, the `stripe` skill will be available in Claude Code. It provides tooling to Generate Stripe integration with checkout and subscriptions.
