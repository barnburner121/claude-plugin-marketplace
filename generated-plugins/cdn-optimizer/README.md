# cdn-optimizer

Generate CDN with cache optimization.

## Installation

```
/plugin install cdn-optimizer@barnburner121
```

## Usage

Once installed, the `cdn-opt` skill will be available in Claude Code. It provides tooling to Generate CDN with cache optimization.
