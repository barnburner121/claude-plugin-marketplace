# test-organizer

Reorganize test suites following best practices.

## Tools Available

- **Read** — Read files from the filesystem
- **Write** — Write files to the filesystem
- **Edit** — Make targeted edits to existing files
- **Bash** — Execute shell commands
- **Grep** — Search file contents with regex
- **Glob** — Find files by pattern matching

## Usage

Invoke the `test-org` skill to Reorganize test suites following best practices. The skill will analyze the relevant codebase context and generate appropriate output.
