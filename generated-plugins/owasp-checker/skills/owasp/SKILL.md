# owasp-checker

Scan codebase against OWASP Top 10 with remediation

## Tools

This skill uses the following tools:

- **Read** — Read files from the filesystem to analyze content
- **Write** — Write new files or overwrite existing files
- **Edit** — Make targeted edits to existing files
- **Bash** — Execute shell commands for scanning, building, and automation
- **Grep** — Search file contents with regex patterns
- **Glob** — Find files by name patterns

## Usage

Invoke this skill by asking Claude to Scan codebase against OWASP Top 10 with remediation.
