# a11y-fixer

Scan HTML/JSX for accessibility issues.

## Tools Available

- **Read** — Read files from the filesystem
- **Write** — Write files to the filesystem
- **Edit** — Make targeted edits to existing files
- **Bash** — Execute shell commands
- **Grep** — Search file contents with regex
- **Glob** — Find files by pattern matching

## Usage

Invoke the `a11y-fix` skill to Scan HTML/JSX for accessibility issues. The skill will analyze the relevant codebase context and generate appropriate output.
