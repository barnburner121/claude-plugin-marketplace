# orm-query-builder

Generate optimized ORM queries.

## Tools Available

- **Read** — Read files from the project
- **Write** — Write new files to disk
- **Edit** — Edit existing files in place
- **Bash** — Run shell commands
- **Grep** — Search file contents with regex
- **Glob** — Find files by pattern

## Usage

Invoke the `orm-query` skill to Generate optimized ORM queries. The skill analyzes your project structure and generates the appropriate code and configuration files.
