# monitoring-stack

Generate monitoring stack setup.

## Tools Available

- **Read** — Read files from the project
- **Write** — Write new files to disk
- **Edit** — Edit existing files in place
- **Bash** — Run shell commands
- **Grep** — Search file contents with regex
- **Glob** — Find files by pattern

## Usage

Invoke the `monitoring` skill to Generate monitoring stack setup. The skill analyzes your project structure and generates the appropriate code and configuration files.
