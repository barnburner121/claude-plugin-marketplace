# api-rate-handler

Generate client-side rate limit handling.

## Installation

```
/plugin install api-rate-handler@barnburner121
```

## Usage

Once installed, the `api-rate` skill will be available in Claude Code. It provides tooling to Generate client-side rate limit handling.
