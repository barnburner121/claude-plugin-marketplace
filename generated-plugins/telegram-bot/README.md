# telegram-bot

Generate Telegram bot with keyboards.

## Installation

```
/plugin install telegram-bot@barnburner121
```

## Usage

Once installed, the `tg-bot` skill will be available in Claude Code. It provides tooling to Generate Telegram bot with keyboards.
