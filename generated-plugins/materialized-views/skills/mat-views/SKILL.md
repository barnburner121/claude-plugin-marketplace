# materialized-views

Generate materialized view definitions.

## Tools Available

- **Read** — Read files from the project
- **Write** — Write new files to disk
- **Edit** — Edit existing files in place
- **Bash** — Run shell commands
- **Grep** — Search file contents with regex
- **Glob** — Find files by pattern

## Usage

Invoke the `mat-views` skill to Generate materialized view definitions. The skill analyzes your project structure and generates the appropriate code and configuration files.
