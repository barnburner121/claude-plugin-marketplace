# css-optimizer

Find unused CSS and optimize stylesheets.

## Tools Available

- **Read** — Read files from the filesystem
- **Write** — Write files to the filesystem
- **Edit** — Make targeted edits to existing files
- **Bash** — Execute shell commands
- **Grep** — Search file contents with regex
- **Glob** — Find files by pattern matching

## Usage

Invoke the `css-opt` skill to Find unused CSS and optimize stylesheets. The skill will analyze the relevant codebase context and generate appropriate output.
