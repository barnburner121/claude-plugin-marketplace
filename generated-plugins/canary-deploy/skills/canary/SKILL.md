# canary-deploy

Generate canary deployment strategies.

## Tools Available

- **Read** — Read files from the project
- **Write** — Write new files to disk
- **Edit** — Edit existing files in place
- **Bash** — Run shell commands
- **Grep** — Search file contents with regex
- **Glob** — Find files by pattern

## Usage

Invoke the `canary` skill to Generate canary deployment strategies. The skill analyzes your project structure and generates the appropriate code and configuration files.
