# runbook-writer

Generate operational runbooks for services.

## Tools Available

- **Read** — Read files from the filesystem
- **Write** — Write files to the filesystem
- **Edit** — Make targeted edits to existing files
- **Bash** — Execute shell commands
- **Grep** — Search file contents with regex
- **Glob** — Find files by pattern matching

## Usage

Invoke the `runbook` skill to Generate operational runbooks for services. The skill will analyze the relevant codebase context and generate appropriate output.
