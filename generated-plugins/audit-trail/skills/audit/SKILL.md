# audit-trail

Generate database audit trail systems.

## Tools Available

- **Read** — Read files from the project
- **Write** — Write new files to disk
- **Edit** — Edit existing files in place
- **Bash** — Run shell commands
- **Grep** — Search file contents with regex
- **Glob** — Find files by pattern

## Usage

Invoke the `audit` skill to Generate database audit trail systems. The skill analyzes your project structure and generates the appropriate code and configuration files.
