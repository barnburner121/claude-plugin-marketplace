# video-transcoder

Generate video transcoding pipeline configs.

## Installation

```
/plugin install video-transcoder@barnburner121
```

## Usage

Once installed, the `video` skill will be available in Claude Code. It provides tooling to Generate video transcoding pipeline configs.
