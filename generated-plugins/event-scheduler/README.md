# event-scheduler

Generate event scheduling with timezones.

## Installation

```
/plugin install event-scheduler@barnburner121
```

## Usage

Once installed, the `evt-sched` skill will be available in Claude Code. It provides tooling to Generate event scheduling with timezones.
